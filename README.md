# Orbis.Nft

A dark, space-themed NFT landing page built with **React + TypeScript + Vite + Tailwind CSS**.

It features CloudFront-served video backgrounds, a custom **liquid-glass** UI
effect, a full-screen grain texture overlay, and a tight Anton / Condiment /
monospace type system.

## Sections

1. **Hero** — full-bleed video, liquid-glass navbar, oversized headline + Condiment accent, social buttons.
2. **About / Intro** — full-bleed video, "Hello! I'm orbis" with a Condiment overlay and a monospace blurb.
3. **Collection** — solid `#010828` grid of liquid-glass NFT cards with looping square videos and rarity overlays.
4. **CTA** — native-aspect video with a right-aligned call-to-action and a vertical social menu.

## Design system

| Token            | Value                              |
| ---------------- | ---------------------------------- |
| Background       | `#010828` (deep navy) — `bg-space` |
| Text             | `#EFF4FF` — `text-cream`           |
| Accent           | `#6FFF00` — `text-neon` / `bg-neon`|
| Headings / nav   | Anton — `font-grotesk`             |
| Cursive accents  | Condiment — `font-condiment`       |
| Body / data      | system monospace — `font-mono`     |

## Getting started

```bash
npm install          # installs deps and (re)generates public/texture.png
npm run dev          # start the Vite dev server
npm run build        # type-check + production build
npm run preview      # preview the production build
```

> `public/texture.png` is generated by `scripts/gen-texture.cjs` (runs
> automatically on `npm install` via the `postinstall` hook). Regenerate it
> any time with `node scripts/gen-texture.cjs`.
